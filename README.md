### Tech Stack -
- Next.js
- MongoDb

### Getting Started -

Create and add below variables in `.env`

```bash
MONGODB_URI=...
USER_EMAIL=...
USER_PASSWORD=...
NEXTAUTH_SECRET=...
```

*Using node version: v20.14.0*

Run the development server:

```bash
npm install
```

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
